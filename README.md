# File Protection Utility

## Description
File Protection Utility is a Python-based tool that converts file content into a protected format and allows it to be restored when required. It helps users safeguard data and understand basic file protection techniques.

## Features
- Protects file content using encoding
- Restores protected files to original format
- Simple and easy-to-use interface
- Supports different file types

## Technologies Used
- Python
- Base64 Encoding

## How to Run

```bash
py file_protection_utility.py
```

## Sample Output

```

2. Restore File

Enter your choice (1/2): 1
Enter file name to protect: sample.txt
Enter protected file name: protected.txt
```
